Quick Start

Download

The original and official Wacara WordPress Theme for Event Organizer is available only on ThemeForest, you can purchase with supported payment methods.

Installation

There are two methods to install the Wacara to your WordPress site, you can install it with or without ftp access, but both method require access to wp-admin dashboard.

Without FTP

  1. Log in to your wp-admin dashboard
  2. Go to Appearance menu and select the sub menu called Themes
  3. Click the Add New button
  4. Click the Upload Theme button
  5. Click the input field which will open your file explorer, and then navigate to your downloaded theme file named wacara.zip
  6. Click the Install Now Button
  7. Wait until the process finished, and then click Activate link
  8. Congratulation, the Wacara theme should be set as your active theme

With FTP

  1. Extract your downloaded theme file which named wacara.zip
  2. Log in to your FTP account
  3. Navigate to site_installation/wp-content/themes/ and upload your extracted theme file here
  4. Log in to your wp-admin dashboard
  5. Go to Appearance menu and select the sub menu called Themes
  6. You should see Wacara Event Organizer in the theme list
  7. Click the Activate button of the Wacara Event Organizer
  8. Congratulation, the Wacara theme should be set as your active theme

Configuration

Your theme is active, but before creating your first event there are several configurations you need to define first.

General Configuration

Default Logo

You need to define the default logo for your website. Here are the steps you need to follow

  1. Log in to your wp-admin dashboard
  2. Go to Wacara Options menu
  3. Simply click Select Image button and don't forget save the changes.

Payment Configuration

Stripe Key

Wacara has manual and automatic payment method, and it uses Stripe as its automatic payment gateway. We are assuming that you are already have Stripe account, if not go create one. Here are the steps to configure your Stripe:

  1. Log in to your wp-admin dashboard
  2. Go to Wacara Options menu and select the sub menu called Stripe Options
  3. Put your publishable key and secret key into the form
  4. Simply click the Save Changes button to save the setting

You may wondering, what are those fields meaning to?, here are the explanations:

  • Sandbox is a useful field to switch between real life payment and testing payment. Make sure to uncheck this on production site.
  • Sandbox Secret Key is field to save the Stripe secret key for sandbox.
  • Sandbox publishable Key is field to save the Stripe publishable key for sandbox.
  • Live Secret Key is field to save the Stripe secret key for production.
  • Live publishable Key is field to save the Stripe publishable key for production.
Bank Information

For manual payment, you have to provide the bank information for your users so they know where to to transfer the payment and you as the admin need to verify the payment manually. Here are the steps to configure your bank information:

  1. Log in to your wp-admin dashboard
  2. Go to Wacara Options menu and select the sub menu called Bank Information
  3. Put the details of your bank accounts. To add more bank accounts simply click the Add Bank button
  4. Click the Save Changes button to save the setting

Important:

Sandbox and Live is not interchangeable, so you must put the correct keys into the correct fields

Event

Event is the main feature of this theme, but this menu itself can not go live without these requirements. As being said, we have to complete these requirements first before creating the event. Those requirements are Header, Locations, Speakers, and Prices.

Actually those requirements aim is really simple that you can not create event before creating headers, locations, speakers, and prices. Go check them all and back to this section once you have done.

First of all, congratulation for completing the creation of headers, locations, speakers, and prices. At this point you will create your first event for real.

Basically, event has three metaboxes for configuring the event as your desire, they are Details, Schedule, and Designs. And we will talk about each one of them.

Details

In this metabox we will separate into three different tabs, General, Rules, and Fields.

General

This tab contains few fields that will be explained below.

  • Date start, is field to configure when the event will be started. This field also has responsibility to the countdown in header section.
  • Single day, as its name, whether we want to create an event that will be held in one day or multiple day. If Single day is checked, it will reproduce another field which is called Time end. The field is used to save information when the event will be ended. In another hand, if Single day is not checked, it will also reproduce a field called Date end. it's opposite from Date start, which is to save the date end time of when the event will be ended.
Rules

This tab only contains one main field, that is Allow register, if you check this option you will allow the user to register to your event. In another hand, if you uncheck this option you will not allow user to register to your event. This option will useful if you are organizing free public event which does not require any registration. Moreover, if you activate the Allow register, it will produce one more field that's called Limit register. Just like its name, the field will add more flexibility to the event whether you want to create an event with limited seats or limited registration time. Currently we can limit the registration by two fields, By participant, and By date.

Fields

In this tab you can configure the fields of your event's registration form.

Basically, every field has three options, whether include the field on the registration form or not, set the field as mandatory, and set the label of the field. But there are two fields that you can not exclude, they are Email and Name.

Schedule

You can display the information of the event's rundown from the start until the finish. You can add as many timeline as you desire, and each timeline has three fields, they are Period, Title, and Content.

Schedule uses repeater field, it means you can create schedule item as many as you want. And in order to add more items click Add Schedule

Design

This metabox has eight different tabs, mostly they are representing the layout of the event's landing page. Those tabs are General, About section, Speakers section, Venue section, Gallery section, Sponsors section, Schedule section, and Pricing section.

General

This tab contains four fields that will be explained below:

  • Section order, this means you an adjust the order of sections in your event's landing page. You can even drop not desired section. But keep in mind, header will always remain in the top.
  • Main logo, you can adjust the logo for each event. and if you leave the field empty your event will use logo of your site which is you have already set it from options.
  • Header, you can design the header differently from one to another, you can learn more about header here.
  • Background, just in case you want use use the same header template from previous created header but you want to modify the background image. But you can always leave it empty to exactly use the same as previous created header.

Before we move further, next part we will walk through the available options for each sections. And all of them have three common fields, they are:

  • Nav title, whether you want to make a shortcut to your section from your nav bar, just put the nav title and it will create a shortcut to the respective section. In another hand, if you leave it empty you won't see the shortcut in the nav bar.
  • Title, maybe you want to name all your sections with your own word, just write the title in this field.
  • Subtitle, this is an additional field you can use to create little bit more explanation to the section.
About Section

Beside the common fields we have mentioned above, this tab has one more field which is Description. You can write a short description about the event and it will be displayed in about section alongside date information and location information.

Speakers Section

This tab also has one more field, which is Speakers, where you can select lists of speakers that you have created before. For more detail about speakers, please refer to this section

Venue Section

Just like two tabs above, this tab also has one field which is Location, where you can select one location you have created before. To learn more about location please refer to this section

Gallery Section

This tab also has one field called Gallery, where you can put bunch of images related to the event, you can select any images.

Sponsors Section

This tab exactly is identical with last tab above, only contain one field for image lists, the only different is the name of the field, which is called Sponsors. This field is used to display list of sponsor logos of the event.

Schedule Section

This tab does not have any field except three common fields above. This tab has its own metabox instead, the metabox called Schedule.

Pricing Section

Last but not least, this tab has major role related to the registration. This tab has one field called Pricing, which is a list of package/tier of the registration. You can select lists of pricing that you have created before. To learn more about pricing, please refer to this section.

Header

Header is a section a top level section of your event's landing page. Normally it tells the visitor a summary about the event. It also has big responsibility about how your event will look like when being accessed. A cool thing about header is, it has various designs to make it unique.

To create a header is relative easy, here are the steps:

  1. Log in to your wp-admin dashboard
  2. Go to Headers menu
  3. Click the Add New button
  4. Put a name for your header, and adjust some configurations
  5. Lasty, click Publish button

You may wondering, what are the meanings of those configurations, here is the explanation.

Header only has one metabox named Detail, and that metabox has two tabs, they are Layout and Content

Layout

Layout has three fields, here they are:

  • Width, is a dropdown field, it has two options, Full-width content and Half-width content. This field is used to set the width of the background image that will be used in header.
  • Scheme, is another dropdown field, it also has two options, Light color and Dark color. If you really want to show off your background image you should select Dark color, because Light color is more suitable if you want to use the background image as an accent. This field is available only if the Width is set to Full-width content.
  • Alignment, is a dropdown field to set the alignment of the Header's content. It has three options, Left, Center, and Right.
Content

This tab is more about what you want to display in the header. It has two fields, here they are:

  • Default image, is a image field to set the background image of the header
  • Countdown, is a checkbox field to set whether you want to show or hide the countdown information in the header.

Tips:

If you choose Half-width content, please use image with a transparent background, may be some illustration or vector image.
